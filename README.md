# Short Agent

Semi-automatic Shorts & Reels content package generator for **DIGI-TV**.

Generates 3–5 production-ready short-video content packages per run for YouTube Shorts, Facebook Reels, Instagram Reels, and TikTok. Packages include scripts, voiceover text, scene prompts, titles, descriptions, hashtags, and upload checklists. No auto-upload. You review and post manually.

---

## 1. Setup

```bash
cd C:\dev\short-agent
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

---

## 2. Create your .env file

Copy the example file and add your OpenAI API key:

```bash
copy .env.example .env
```

Open `.env` and replace the placeholder:

```
OPENAI_API_KEY=sk-...your-real-key-here...
```

> The `.env` file must be in `C:\dev\short-agent`. It is already in `.gitignore` and will not be committed.

---

## 3. Install requirements

```bash
pip install -r requirements.txt
```

---

## 4. Run the server

```bash
uvicorn main:app --reload
```

## 7. Example API request

```bash
curl -X POST http://127.0.0.1:8000/generate \
  -H "Content-Type: application/json" \
  -d '{
    "channel_name": "DIGI-TV",
    "content_theme": "Digital infotainment, technology education, AI, automation, mobility, engineering, and social impact",
    "content_pillars": [
      "AI tools and automation",
      "future mobility and self-driving technology",
      "engineering and infrastructure",
      "digital business and creator economy",
      "social impact of technology"
    ],
    "number_of_packages": 5,
    "duration_seconds": 60,
    "language": "English",
    "platforms": ["YouTube Shorts", "Facebook Reels", "Instagram Reels", "TikTok"],
    "extra_instruction": "Find relevant trending or evergreen ideas suitable for DIGI-TV.",
    "save_output": true
  }'
```

---

## Output files

When `save_output` is `true`, two files are saved in the `outputs/` folder:

- `outputs/YYYYMMDD_HHMMSS_short_packages.json` — structured JSON
- `outputs/YYYYMMDD_HHMMSS_short_packages.md` — human-readable Markdown

---

## Notes

- This tool does **not** upload videos. It only generates content packages.
- Review every package before publishing.
- All content is generated by OpenAI GPT-4o based on DIGI-TV guidelines.
- Topics are limited to technology, AI, automation, mobility, engineering, digital business, and social impact — no politics, gossip, or unrelated entertainment.
